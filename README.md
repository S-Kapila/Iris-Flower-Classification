# 🌸 Iris Flower Classification

A simple and beginner-friendly **Machine Learning project** that classifies iris flowers into different species based on their physical characteristics. This project demonstrates how classification algorithms can learn patterns from data and make predictions on unseen samples.

## 📌 Project Overview
The model is trained using the famous Iris dataset, which contains measurements of iris flowers. Using these measurements, the model predicts the correct species of the flower.

The three classes in the dataset are:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

Each sample contains four features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
